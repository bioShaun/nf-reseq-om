简介

获得样品后，如下图所示流程进行试验和数据处理，得到最终的分析结果
1) 对样品DNA进行随机打断建库.
2) 对捕获的文库使用Illumina测序平台进行测序.
3) 获得下机数据后使用GTAK最佳实践流程进行数据的标准call SNP分析得到vcf文件.
4) 使用个性化分析脚本进行数据的统计分析.

实验流程

提取每个个体的基因组DNA并随机打断，电泳回收所需长度的DNA片段，并加上接头进行cluster制备，最后在Hiseq平台上机测序。

图2：建库流程

数据分析流程

运用BWA软件将测序得到的reads与参考基因组序列进行比对。 根据比对结果，计算出相对于参考基因组的测序深度和覆盖度。 所测基因组的SNPs、InDels和结构变异SVs分别用 GATK、samtools、breakdancer等软件进行检测。 当有参考基因组的注释文件，可以分别在SNP、InDel和SV的数据集的基础上进行注释。

我们采用 Broad Institute 开发的变异检测流程 GATK，参考 GATK 的推荐流程 (Best Practices for Germline SNP & Indel Discovery in Whole Genome and Exome Sequence)， 对过滤后的测序序列进行参考基因组的比对分析. 主要流程如下：
(1) Map to reference (BWA default parameters)
(2) Mark Duplicates (Picard)
(3) Realign Indels (Picard)
(4) Recalibrate Base (Picard)
(5) Call Variants (GATK)
(6) VCF Quality filter (GATK standard)
(7) Annotating variants (snpEff/Python Script)
